## Quick start

1. Clone the project.

2. Rename `.env.example` to `.env` and fill it with your environment values.

3. Install dependencies:

   ```bash
   go mod download
   ```

4. Run project:

   ```bash
   go run .
   ```

- *If you have air package for live reloading, run the project with:*

   ```bash
   air
   ```
